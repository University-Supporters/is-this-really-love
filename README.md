# Is This Really Love? 💘 (이게 정말 사랑일까?)

> **"지연이... 아니면 내 사랑❤️? 당신의 폰으로 걸려오는 집착의 목소리를 체험해보세요."**

단순히 글자로만 배우는 인권 교육은 끝났음! 이 프로젝트는 실제 상황과 유사한 **체험형 웹 시뮬레이션**을 통해 스토킹과 교제폭력의 위험성을 뼛속까지 느끼게 해줌. 칙칙한 포스터 대신, 당신의 브라우저에서 직접 '집착'을 마주하며 인식을 전면 개편(Awareness Transition)하는 것이 목표임.

---

## 🚀 주요 기능 (Main Features)

### 1. 극사실주의 수신 전화 시뮬레이션 📞
- **리얼한 UI & 진동**: 실제 iOS 환경을 그대로 옮겨온 듯한 수신 화면과 브라우저 진동(Vibration API)을 통해 심리적 압박감을 생생하게 전달함.
- **배경 이미지 연동**: 전화가 오는 순간, 발신자의 사진이 배경에 큼지막하게 뜨며 당신의 일상을 잠식하는 공포를 시각화함.

### 2. 성별 맞춤형 & 랜덤 발신자 시스템 👥
- **당신의 성별은?**: 여성/남성 선택에 따라 각기 다른 상황이 펼쳐짐.
- **6인의 빌런(녹음본)**: 각 성별마다 3명씩, 총 6개의 랜덤 시나리오가 준비되어 있음. 시작할 때마다 누가 당신의 폰을 울릴지 모르는 쫄깃한 긴장감.

### 3. 몰입도 높은 통화 인터페이스 🎧
- **실제 녹음본 재생**: 통화를 수락하는 순간, 준비된 실제 음성 파일이 재생되며 상황 속으로 당신을 끌어들임.
- **실시간 파형 & 타이머**: 통화 중임을 알려주는 역동적인 파형(Visualizer)과 초 단위로 흐르는 시간을 보며 숨 막히는 순간을 체험함.

### 4. 프리미엄 글래스모피즘 디자인 (Web Transition) 💻
- **Tailwind CSS v3 기반**: 촌스러운 기본 테마는 잊으셈. 세련된 그라데이션과 투명한 유리 질감(Glassmorphism) UI로 눈이 즐거운 고퀄리티 체험을 제공함.
- **부드러운 화면 전환**: 페이지가 바뀔 때마다 페이드 인/아웃 효과가 적용되어 고급스러운 앱 사용 경험을 선사함.

### 5. 인권 인식 개선 & 캠페인 연계 🚨
- **팩트 폭격 경고**: 체험이 끝나면 이것이 '사랑'이 아닌 '폭력'임을 명확히 짚어줌.
- **인스타 다이렉트 연동**: 클릭 한 번으로 '인권 서포터즈' 공식 인스타그램으로 이동하여 더 많은 정보와 캠페인에 참여할 수 있음.

## ⚙️ 시작하기 (Getting Started)

체험 준비는 간단함. 아래 명령어로 세련된 웹 서버를 띄워보셈.
1. 의존성 설치함.
   ```bash
   npm install
   ```
2. 바로 실행 때림.
   ```bash
   npm run dev
   ```
3. 브라우저에서 `http://localhost:5173` 접속하면 시뮬레이션 시작!

## 🎮 시뮬레이션 흐름 (Simulation Flow)

1. **성별 선택**: 당신의 성별을 선택함. (이에 맞춰 최적화된 시나리오가 준비됨)
2. **수신 대기**: "시작하기"를 누르는 순간, 누군가로부터 전화가 걸려오며 폰이 진동함.
3. **선택의 갈림길**: 전화를 받을 것인가, 거절할 것인가? (하지만 거절해도 폭력의 진실은 마주하게 됨)
4. **진실 마주하기**: 통화 종료 후 나타나는 안내 메시지를 통해 우리가 사랑이라 착각했던 행동들의 실체를 파악함.
5. **캠페인 참여**: 인권 서포터즈 인스타를 팔로우하고 더 건강한 사랑을 배우러 감.

## 🔧 최근 업데이트 이력 (v2.0 대격변 패치)

- **Tailwind CSS 전면 전환**: 투박했던 스타일을 싹 빼고 트렌디한 글래스모피즘 디자인으로 재탄생함.
- **랜덤 발신자 시스템**: 성별 버전별로 3명씩, 총 6가지의 다른 목소리를 만날 수 있게 됨.
- **배경 사진 연동**: 전화가 올 때 발신자 사진이 배경에 흐릿하게 깔리도록 하여 몰입감을 극대화함.
- **상대 경로 핫픽스**: 배포 환경에서 이미지가 안 뜨던 고질적인 버그를 완벽히 고침.
- **UI 직관성 개선**: 불필요한 OS 선택 단계를 없애고 성별 선택 위주로 흐름을 단순화함.
